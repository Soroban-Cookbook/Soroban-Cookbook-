# Soroban Cookbook Guides

Step-by-step guides to help you build, test, and deploy smart contracts on Soroban.

## 📖 Available Guides

### [Getting Started](./getting-started.md)
Learn how to set up your Soroban development environment, including Rust, the Soroban CLI, and VS Code.

### [Testing Guide](./testing.md)
A comprehensive guide to testing Soroban contracts, from unit tests to advanced integration testing.

### [Deployment Guide](./deployment.md)
Learn how to deploy your smart contracts to Stellar's Testnet and Mainnet, including transaction fee management.

### [Ethereum to Soroban](./ethereum-to-soroban.md)
A translation guide for Ethereum developers, comparing Solidity patterns with their Soroban equivalents.

## 📋 Planned Guides

- **Gas Optimization Guide** - Advanced techniques for minimizing contract resource usage.
- **Security Best Practices** - A checklist for securing your Soroban smart contracts.
- **Frontend Integration** - Connecting your web applications to Soroban contracts using the Stellar SDK.
- **State Archival & TTL** - Deep dive into managing contract state and time-to-live.

## 🚀 Learning Path

1. Start with **[Getting Started](./getting-started.md)** to set up your environment.
2. Follow the **[Basics](../examples/basics/)** examples to learn core concepts.
3. Use the **[Testing Guide](./testing.md)** as you develop your own contracts.
4. Refer to the **[Deployment Guide](./deployment.md)** when you're ready to go live.
