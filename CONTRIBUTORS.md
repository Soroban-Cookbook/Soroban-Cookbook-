## Tier Legend

| Badge | Tier | Description |
|-------|------|-------------|
| 🌱 | Newcomer | First accepted PR |
| ⭐ | Contributor | 3+ accepted PRs or 1 high-impact contribution |
| 🔥 | Regular Contributor | 10+ accepted PRs or sustained multi-phase contributions |
| 💎 | Core Contributor | Deep expertise, mentorship, sustained leadership |
